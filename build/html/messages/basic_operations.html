Basic Operations¶

Creating a message¶

POST /my-app/queues/foo/bar¶

Request Body

{
    "name" : "test"
}

Response Body

{
    "status" : "ok",
    "path" : "/foo/bar/",
    "queue" : "1c184f38-9134-3400-b802-81315d9e7388",
    "messages" : [
        {
            "name" : "test",
            "timestamp" : 1314034739489,
            "uuid" : "9e33d9e5-cce5-11e0-9af9-9227e40e3559"
        }
    ],
    "last" : "9e33d9e5-cce5-11e0-9af9-9227e40e3559",
}
